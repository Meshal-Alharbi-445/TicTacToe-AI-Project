# TicTacToe-AI-Project

This project is an implementation of a smart Tic-Tac-Toe game powered by the **Minimax** algorithm, optimized with **Alpha-Beta Pruning**.

## Project Description
- **The Problem:** In competitive games, players struggle to process all possible future moves to ensure a win.
- **The Solution:** An AI Agent that analyzes the game board and calculates the optimal move to guarantee the best outcome.
- **Algorithm:** **Alpha-Beta Pruning** is used to prune unnecessary search branches, significantly improving decision-making speed without sacrificing accuracy.

## Requirements
- Python 3.x

## How to Run
1. Run `main.py` using a Python interpreter.
2. Enter the board position index (0-8) to play against the AI.
